\doxysubsection{Detailed Description}
A functor that reports error to supporting system. Invoked similar to printf.

Usage\+: \mbox{\hyperlink{classtflite_1_1_error_reporter}{Error\+Reporter}} foo; foo.\+Report(\char`\"{}test \%d\char`\"{}, 5); or va\+\_\+list args; foo.\+Report(\char`\"{}test \%d\char`\"{}, args); // where args is va\+\_\+list

Subclass \mbox{\hyperlink{classtflite_1_1_error_reporter}{Error\+Reporter}} to provide another reporting destination. For example, if you have a G\+UI program, you might redirect to a buffer that drives a G\+UI error log box.
